  1. Briefly Describe any 3 web scripting languages.

    2. There are 3 web scriping languages namely:


    • Javascrip (JS): this is a versatile scripting language primarily used for client side scripring.
      Developers use it to add interactive elements to the website and create a dynamic content
      It supports event handling and is widely supported by all major web browsers
    • PHP (Hypertext preprocessor):This is a server side scripting language designed for web developers
      It is embedded within HTML and executed on the server to generate dynamic web pages.
      It handles forms, manages cookies, interacts with databases. etc
      Used to create content management systems, ecomerce platforms and other dynamic web applications
    • Python: This is a high level programming language known for its simplicity and readability.
      Supports Object Oriented and functional programming paradigms and known for its clear syntax
      It is used for building web applications, data driven websites and perfoming tasks like Data analysis and machine learning within web environments.
